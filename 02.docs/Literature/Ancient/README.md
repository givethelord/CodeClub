# 朝代歌

    尧舜禹相传
    夏商与西周
    东周分两段
    春秋和战国
    一统秦两汉
    三分魏蜀吴
    两晋前后延
    南北朝并立
    隋唐五代传
    宋元明清后
    皇朝至此完
