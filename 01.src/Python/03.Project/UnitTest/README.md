# UnitTest

## 目录结构说明:

1. TestCase(存放文本以及自动化测试用例) 
    1. 00-特性1
    2. 01-特性2
    3. 02-特性3
2. Resource(存放关键字以及资源引用文件)
3. Variables(存放公共变量和环境变量)
    1. Common(存放公共环境变量)
    2. Env(存放各个环境的变量)
4. TestFiles(存放测试使用的数据文件)
5. TestContent(被测模块)
6. Report(测试报告)
7. unitest_main.py(执行入口)
