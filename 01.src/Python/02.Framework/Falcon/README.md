# Falcon

　　Falcon是一个构建云API的高性能Python框架，它鼓励使用REST架构风格，尽可能以最少的力气做最多的事情

特性
1. 通过URI模板和资源类的路由
2. 通过请求和响应类访问headers和bodies
3. 通过异常基类响应HTTP错误等等
