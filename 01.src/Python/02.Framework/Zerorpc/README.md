# Zerorpc

　　Zerorpc是一个基于ZeroMQ和MessagePack开发的远程过程调用协议（RPC）实现。和 Zerorpc 一起使用的 Service API 被称为 zeroservice。Zerorpc 可以通过编程或命令行方式调用。

它允许你：
1. 不用修改代码即可显露python模块
2. 通过命令行远程调用这些模块
