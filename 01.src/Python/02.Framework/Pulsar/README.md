# Pulsar

　　Pulsar是一个事件驱动的并发框架，有了pulsar，你可以写出在不同进程或线程中运行一个或多个活动的异步服务器。

附带以下功能

1. Socket服务器
2. WSGI服务器
3. JSON-RPC
4. Web Sockets
5. 任务队列
6. Shell
7. 测试包
8. django集成
